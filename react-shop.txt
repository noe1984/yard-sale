-----SASS
npm i mini-css-extract-plugin css-loader style-loader sass sass-loader -D
In webpack 
    const MiniCssExtractPlugin = require('mini-css-extract-plugin')
In rules
    {
        test:/\.s[ac]ss$/i,
        use:[
            "style-loader",
            "css-loader",
            "sass-loader"
        ]
    }
In plugins
    new MiniCssExtractPlugin({
        filename: ["[name"].css
    }),
At the end add 
    devServer: {
        static: {
            directory: path.join(__dirname, 'public'),
            },
        compress: true,
        port: 3005,
    }
